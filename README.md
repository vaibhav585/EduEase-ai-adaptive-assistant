# 🧠 AI Adaptive Assistant for Neurodivergent Learners

An intelligent, inclusive learning platform designed to support neurodivergent individuals (such as those with ADHD, dyslexia, and autism) through personalized, adaptive AI-driven assistance.

---

## 🚀 Overview

This project is an AI-powered assistant that enhances learning accessibility by adapting content, interaction styles, and feedback based on user needs. It integrates multiple assistive technologies to make learning more engaging, understandable, and inclusive.

The system focuses on improving:
- 🧩 Comprehension
- 🎯 Focus & attention
- 📖 Readability
- 🌍 Accessibility across languages

---

## ✨ Key Features

### 🤖 AI Chatbot
- Context-aware conversational assistant
- Helps users understand concepts interactively

### 📖 Text Simplification
- Converts complex content into easy-to-read format
- Improves readability for dyslexic users

### 🌐 Translation Support
- Multi-language support for better accessibility

### 🧠 Personalized Recommendations
- Suggests learning strategies based on user behavior

### 📊 Analytics Dashboard
- Tracks engagement and learning progress
- Provides insights for improvement

### 🧪 Quiz Generation
- Auto-generates quizzes from learning material
- Reinforces understanding

### 👁️ Eye Tracking (Focus Detection)
- Monitors user attention
- Enhances adaptive learning experience

### 🔊 Accessibility Tools
- Text-to-Speech (TTS)
- Speech-to-Text (STT)
- Visual step-by-step learning
